Floating points numbers
scientific notation: 2.6302 x 10^30

type 		storage		precision		exponent
Float 		4 bytes		7 digits		+- 38
double 		8 bytes		16 digits		+- 308


float x = 1.0f; //makes it float
double y = 1.0;//defaults to double

Representation 

try not to use x == y with doubles
try to use, instead, fabs(x - y) < 0.000001; 
this ensures that there are no errors due to rounding errors, or 
number representation errors

0.0 and 1.0 can be represented exactly so
if (x == 0.0) or if (x == 1.0) might be okay..depends on the scenario
if x is a calculated value then it might not be okay

if using fabs(); must #include <math.h>
